Описание архива

Архив содержит HTML-код формы обратной связи и необходимые для его работы ресурсы (изображения, стили, скрипты). Он собран автоматически с помощью конструктора форм обратной связи на Заходи-ка.ру. Инструкция по установке формы на сайт находится в INSTALL.html.

Обратная связь

Над улучшением конструктора и расшинением его функциональности ведется постоянная работа. Сообщить об ошибке или оставить запрос на реализацию новой функциональности вы можете на странице проекта в системе Реформал. По остальным вопросам пишите через форму обратной связи на Заходи-ка.ру.
