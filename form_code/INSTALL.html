Установка

  1. Скопируйте фрагменты кода формы (обозначены комментариями) из тестовой страницы index.html на страницу своего сайта.
  2. Поместите папку с ресурсами zff в корень своего сайта, чтобы ресурсы были доступны по ссылке http://your-site.ru/zff/resource.ext. При размещении папки с ресурсами в другом месте поменяйте вручную пути к ресурсам в коде формы и в файлах CSS-стилей.

Регистрация

После установки формы ее необходимо зарегистрировать на сайте Заходи-ка.ру, чтобы наша система знала, как обрабатывать данные формы и на какой адрес электронной почты отправлять письма. Регистрацию нужно выполнять всякий раз при установке новой формы, или изменении структуры существующей. При отправке письма система автоматически предложит зарегистрировать форму, если в этом будет необходимость. Запустить регистрацию вручную можно со страницы регистрации.
